General Description

Designed for students beginning to learn grammar and elementary arithmetic including addition, subtraction, multiplication, and division as well as an additional category of prime numbers.

The grammar portion of the game includes three category choices that students can select which are nouns, verbs, or adjectives.

Students have the option of choosing from three difficulty levels for both game types which are easy, intermediate, and hard. Game variables can also be switched to alter how the game is played such as player speed as well as the number of possible answers.

Game Details

For math based problems, the number range will vary for each problem type depending upon the difficulty level chosen. The following list provides the problem type followed by the number range for easy, intermediate, nd hard difficulty:

  • Addition (0-6, 0-9, 0-12)
  • Subtraction (0-6, 0-12, 0-20)
  • Multiplication (0-6, 0-9, 0-12)
  • Division (0-10, 0-30, 0-60)
  • Prime Numbers (0-7, 0-19, 0-31)

For grammar based problems, the harder the difficulty selected, the greater the number of syllables for the part of speech. The following list provides the part of speech followed by examples for easy, intermediate, and hard difficulty.

  • Nouns (car, baseball, afternoon)
  • Verbs (run, bubble, concentrate)
  • Adjectives (red, boring, embarrased)
